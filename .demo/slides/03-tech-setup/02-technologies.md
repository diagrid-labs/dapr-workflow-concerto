# Software technologies used

- [.NET 10](https://dotnet.microsoft.com/download/dotnet/10.0) / ASP.NET Core (Minimal API)
- [Dapr](https://dapr.io) (Workflow, Service Invocation)
- [Docker](https://www.docker.com/products/docker-desktop/) (for Dapr services & Redis)
- [Server-Sent Events](https://developer.mozilla.org/docs/Web/API/Server-sent_events) (SSE)
- [P5.js](https://p5js.org/) (Creative coding framework)
- [Web MIDI API](https://developer.mozilla.org/docs/Web/API/Web_MIDI_API) & [Web Audio API](https://developer.mozilla.org/docs/Web/API/Web_Audio_API) (supported by all major browsers)
- [MIDI](https://en.wikipedia.org/wiki/MIDI) - Musical Instrument Digital Interface

_90% of this stack is open source or an open standard._
